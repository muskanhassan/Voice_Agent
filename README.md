# AI Voice Assistant

A smart AI voice assistant built using FastAPI, Groq, MongoDB, and Speech Recognition.

## Features

- AI Chat Assistant
- Speech Recognition
- Male & Female Voice
- Text to Speech
- MongoDB Transcript Storage
- Modern UI

## Tech Stack

- FastAPI
- Groq API
- MongoDB
- HTML
- CSS
- JavaScript

## Installation

Clone repository:

```bash
git clone <repo-link>
```

Go inside folder:

```bash
cd voice-agent
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run server:

```bash
uvicorn app.main:app --reload
```

Open browser:

```text
http://127.0.0.1:8000
```
